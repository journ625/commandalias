# CMA
Lightweight command alias plugin, on Purpur, 1.18.2 Java 17,

# COMMANDS
/cma create "COMMAND" "ALIAS"
- Example: command is "essentials reload" and "esr" is the alias. When you type /esr, Essentials will reload.
/cma list
- You list all of the commands.
/cma edit "ALIAS" "NEW ALIAS"
- You can edit your aliases. So you can edit "esr" to "er", so you will only need to type /er next time.
/cma remove "ALIAS"
- You can delete the created aliases. So you can't use /er anymore.
